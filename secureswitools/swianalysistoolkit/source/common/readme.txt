This folder contains files which are common to all the Signing Diagnostic Tools:

1.osinterface.cpp and osinterface.h

	These are the abstract classes which are used in order to support portability between linux and windows.The current implementation supports windows platform.

2.streamreader.cpp and streamreader.h

	These are common between dumpswicertstoretool and dumpswiregistrytool.

3.ucmp.h and ucmp.cpp

	Provides Standard Compression Scheme for Unicode.
